# MagicSaibao 魔法赛宝

一款简单蹦床游戏的Demo示例程序，用于帮助新手熟悉Magic.h图形框架的开发方式。

目前实现了如下特性：

- 蹦床效果：带有重力加速度，有真实感的物理计算
- 鼠标侦测：踏板跟随鼠标移动
- 状态判断：当赛宝触碰到边缘，游戏结束
